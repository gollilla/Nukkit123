Nukkit123とは？ / About Nukkit123
----------------------------------

Nukkit123は日本MCBEサーバー「123鯖」の開発主(kuro)の個人開発停止に伴い出来上がったプロジェクトです。

123鯖で培われてきた良い要素を Nukkitの派生形として活かしていく事が目的です。
開発に関しては共同開発となりますが、大前提として123鯖に魅力を感じていた方の参加をお待ちしております。

123鯖は独自要素が多く、機能も多岐にわたるため他に反映が難しい派生形になるでしょう。

内容詳細は公式WEBで < http://123saba.fun/wiki/ >

※お願い事
  Nukkit123を使用してサーバーを立ち上げられる際は123鯖派生形である事をWEBや鯖機能のどこかで示してください。

＜英文＞
Nukkit 123 is a project completed in connection with the suspension of individual development of the developer (kuro) of the Japan MCBE server "123 mackerel".

123 The aim is to make good use of the good elements cultivated in mackerel as a derivative form of Nukkit.
Regarding development, it will be joint development, but we are waiting for the participation of those who felt charm to 123 mackas as a premise.

123 Since mackerel has many unique elements and its functions are diverse, it will be a derived form which is difficult to reflect on others.

   When you can start up the server using Nukkit 123, please show me that it is a 123 server derived type somewhere in WEB or mackerel function.
